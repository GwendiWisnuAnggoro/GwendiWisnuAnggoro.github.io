The Keyboard Keys Used To Run Are
W (move up), A (move left), S (moves down), D (moves right).
Before Starting the Game Select One of the W,A,S,D Keys.
Then the snake will run in all directions when you click the first time, after that play as usual!!, After that play as usual !!,
